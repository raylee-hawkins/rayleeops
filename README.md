# rayleeops

Source for [rayleeops.com](https://rayleeops.com) / The Ledger.

The Ledger is the public review layer for AI-assisted technical work — the contested-claims surface where claims are examined in public, against the evidence, on separate evaluation axes by reviewers blind to one another, before they are allowed to close.

This is **not** a portfolio. Closed claims live at [hawkinsops.com](https://hawkinsops.com); each paired case study there links back to the Ledger entry that contested it. Claims that fail review stay here, including retractions.

## Surface routing

- **Closed claims / proof surface:** [hawkinsops.com](https://hawkinsops.com)
- **Contested claims / public review layer:** this site / The Ledger
- **Methodology paper:** [methodology.md](./methodology.md)
